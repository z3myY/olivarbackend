Gestión Olivar Backend

Frontend en https://tiendafrontend.now.sh

API RESTful

OLIVARES

GET /api/olivares Obtener todos los olivares.
POST /api/olivares Añadir un olivar.
GET /api/olivares/{id} Obtener un olivar.
UPDATE /api/olivares/{id} Modificar un olivar.
DELETE /api/olivares/{id} Eliminar un olivar.

AGRICULTORES

GET /api/agricultores Obtener todos los agricultores.
POST /api/agricultores Añadir un agricultor.
GET /api/agricultores/{id} Obtener un agricultor.
UPDATE /api/agricultores/{id} Modificar un agricultor.
DELETE /api/agricultores/{id} Eliminar un agricultor.
